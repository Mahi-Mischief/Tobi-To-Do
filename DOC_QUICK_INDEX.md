## ğŸ’¡ Quick Facts

### Project
- **Name:** Tobi To-Do
- **Type:** AI-powered productivity app
- **Target:** Students & professionals
- **Status:** MVP ready for development

### Tech Stack
- **Frontend:** Flutter 3.0+, Riverpod
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Auth:** JWT + Firebase

### Architecture
- **5 Core Pillars:** Plan, Execute, Improve, Become, Assist
- **5 App Tabs:** Dashboard, Plan, Focus, Growth, Profile
- **14+ API Endpoints**
- **5 Database Tables**

### Features
- Task management (CRUD)
- Habit tracking
- Goal setting
- Focus sessions (Pomodoro)
- Analytics & reports
- Gamification (XP, streaks)
- Dream Me (future self alignment)

### AI Strategy
- **Primary:** Hugging Face Mistral-7B (free tier)
- **Approach:** Logic-first, AI when needed
- **Cost:** Minimal (batch calls, cached)
